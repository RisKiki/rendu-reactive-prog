# rendu-reactive-programming-ing3

## Clone
Pour clone : 
```git clone https://github.com/RisKiki/rendu-reactive-programming-ing3```

## Lancer
Pour lancer : docker 
```docker-compose up -d --build```

## Ports
L'ui écoute sur localhost:8080
L'api écoute sur localhost:5000
